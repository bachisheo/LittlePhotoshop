Color  LightColor,
Color  DarkColor 
)

Конструктор фильтра для цветной бинаризации
Данный фильтр определяет новое значение для цвета пикселя по формуле:

\[ newColor(x) = \left\{ \begin{array}{lr} LightColor & \text{if } l(x) > limit,\\ DarkColor & \text{if } l(x) <= limit. \\ \end{array} \right.\\ \]

где:
newColor(x) - новый цвет,
limit - порог бинаризации,
x - исходный цвет пикселя,
l(x) - яркость цвета х,
LightColor - цвет точек, яркость которых выше порога,
DarkColor - цвет точек, яркость которых ниже или равна порогу бинаризации.

Аргументы
levelПорог бинаризации
LightColorЦвет точек, яркость которых выше порога
DarkColorЦвет точек, яркость которых ниже или равна порогу бинаризации
