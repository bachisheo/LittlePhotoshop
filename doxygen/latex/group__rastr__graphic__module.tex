Конструктор фильтра для цветной бинаризации ~\newline
 Данный фильтр определяет новое значение для цвета пикселя по формуле\+: 

\[ newColor(x) = \left\{ \begin{array}{lr} LightColor & \text{if } l(x) > limit,\\ DarkColor & \text{if } l(x) <= limit. \\ \end{array} \right.\\ \]

где\+:~\newline
new\+Color(x) -\/ новый цвет,~\newline
limit -\/ порог бинаризации,~\newline
x -\/ исходный цвет пикселя,~\newline
l(x) -\/ яркость цвета х,~\newline
Light\+Color -\/ цвет точек, яркость которых выше порога,~\newline
Dark\+Color -\/ цвет точек, яркость которых ниже или равна порогу бинаризации.~\newline
Пример работы фильтра (порог бинаризации 168)\+:

\hline

\end{DoxyParams}
